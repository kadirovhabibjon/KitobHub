# KitobHub

KitobHub — FastAPI, React, PostgreSQL, Redis, Kafka va Docker Compose asosida quriladigan onlayn kitob do‘koni loyihasi.

## Maqsad

Bu loyiha o‘quv va portfolio uchun yaratilmoqda. Loyiha orqali Python backend, mikroservislar, API Gateway, database migration, cache, Kafka event, frontend integratsiya va CI/CD amaliyotlari o‘rganiladi.

## Asosiy texnologiyalar

- Backend: Python, FastAPI
- Frontend: React, TypeScript, Vite
- Database: PostgreSQL
- ORM: SQLAlchemy
- Migration: Alembic
- Cache: Redis
- Message broker: Apache Kafka
- API Gateway: NGINX
- Tests: pytest
- Lint/Format: Ruff
- CI/CD: GitHub Actions

## Dastlabki servislar

- catalog-service
- order-service
- tools-service
- notification-worker
- api-gateway
- frontend
